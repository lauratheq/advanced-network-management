== Description ==

This plugin helps managing WordPress plugins accross a multisite network. It provides advanced visibility options and provides the possibility to force de-/activation.

Currently available languages:
* English
* German

Help contributing to this project on [github](https://github.com/lauratheq/advanced-plugin-management/)

== Screenshots ==

1. Shows the manage plugin link on the plugin overview in the network admin
2. The manage plugin screen

== Installation ==
1. Ensure you have a working multisite
2. Clone this repository to `wp-content/plugins/advanced-plugin-management`
3. Head to Network Admin > Plugins
4. Network activate the Plugin \"Advanced Plugin Management for Multisite\"
